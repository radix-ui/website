### Installation

Install the component from your command line.

```bash
npm install @radix-ui/react-context-menu
```

Then import the component in your component file.

```jsx
// ContextMenu.tsx
import * as ContextMenuPrimitive from '@radix-ui/react-context-menu';
```

### Basic example

```jsx live manual
// ContextMenu.tsx
// import * as ContextMenuPrimitive from '@radix-ui/react-context-menu';

const MenuPopper = styled(ContextMenuPrimitive.ContextMenuPopper, {
  minWidth: 130,
  backgroundColor: 'white',
  borderRadius: 6,
  padding: 5,
  boxShadow:
    'hsla(206,22%,7%,.15) 0px 5px 15px -5px, hsla(206,22%,7%,.1) 0px 5px 10px -15px',
});

const MenuItem = styled(ContextMenuPrimitive.ContextMenuItem, {
  fontSize: 13,
  padding: '5px 10px',
  borderRadius: 3,

  '&[data-disabled]': { color: 'slategray' },

  ':focus': {
    outline: 'none',
    backgroundColor: 'dodgerblue',
    color: 'white',
  },
});

const MenuSeparator = styled(ContextMenuPrimitive.ContextMenuSeparator, {
  height: 1,
  backgroundColor: 'gainsboro',
  margin: '5px 10px',
});

const MenuArrow = styled(ContextMenuPrimitive.ContextMenuArrow, {
  fill: 'white',
});

const TriggerArea = styled(ContextMenuPrimitive.ContextMenuTrigger, {
  display: 'block',
  textAlign: 'center',
  padding: 100,
  backgroundColor: 'slategray',
});

render(
  <ContextMenuPrimitive.Root>
    <TriggerArea>Right click anywhere here</TriggerArea>
    <MenuPopper sideOffset={-5}>
      <MenuItem onSelect={() => alert('undo')}>Undo</MenuItem>
      <MenuItem onSelect={() => alert('redo')}>Redo</MenuItem>
      <MenuSeparator />
      <MenuItem disabled onSelect={() => alert('cut')}>
        Cut
      </MenuItem>
      <MenuItem onSelect={() => alert('copy')}>Copy</MenuItem>
      <MenuItem onSelect={() => alert('paste')}>Paste</MenuItem>
      <MenuArrow />
    </MenuPopper>
  </ContextMenuPrimitive.Root>
);
```

### API Reference

#### ContextMenu

`ContextMenu` is the root component.

#### ContextMenuTrigger

`ContextMenuTrigger` is the area that triggers the `ContextMenu`. Wrap this around the thing you want your `ContextMenu` to open when right-clicking (or using the relevant keyboard shortcuts).

#### ContextMenuPopper

`ContextMenuPopper` is the component that pops out in an open `ContextMenu`.

#### ContextMenuGroup

Used to group multiple `ContextMenuItem`.

#### ContextMenuLabel

Uused to render a label. It won't be focusable using arrow keys.

#### ContextMenuItemIndicator

`ContextMenuItemIndicator` will render when the parent `ContextMenuCheckboxItem` or `ContextMenuRadioItem` is checked. You can style this element directly, or you can use it as a wrapper to put an icon into, or both.
