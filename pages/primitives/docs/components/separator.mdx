### Installation

Install the component from your command line.

```bash
npm install @radix-ui/react-separator
```

Then import the component in your component file.

```jsx
// Separator.tsx
import * as SeparatorPrimitive from '@radix-ui/react-separator';
```

### Basic example

Create your styled separator component from the separator primitive.

```jsx live manual
// Separator.tsx
// import * as SeparatorPrimitive from '@radix-ui/react-separator';

const Separator = styled(SeparatorPrimitive.Root, {
  backgroundColor: 'gainsboro',
  height: 1,
});

render(<Separator />);
```
