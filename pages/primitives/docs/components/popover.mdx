### Installation

Install the component from your command line.

```bash
npm install @radix-ui/react-popover
```

Then import the component in your component file.

```jsx
// Popover.tsx
import * as PopoverPrimitive from '@radix-ui/react-popover';
```

### Basic example

```jsx live manual
// Popover.tsx
// import * as PopoverPrimitive from '@radix-ui/react-popover';
// import { Button } from '@radix-ui/ds-button'

const PopoverContent = styled(PopoverPrimitive.Content, {
  borderRadius: 3,
  padding: '20px',
  fontSize: 14,
  backgroundColor: 'gainsboro',
  color: 'black',
});

render(
  <PopoverPrimitive.Root>
    <PopoverPrimitive.Trigger as={Button}>Trigger</PopoverPrimitive.Trigger>
    <PopoverPrimitive.Popper>
      <PopoverContent>
        <h3>Popover content</h3>
        <p>Are you sure you wanna do this?</p>
        <PopoverPrimitive.Close as={Button}>Yes</PopoverPrimitive.Close>
      </PopoverContent>
      <TooltipPrimitive.Arrow style={{ fill: 'gainsboro' }} />
    </PopoverPrimitive.Popper>
  </PopoverPrimitive.Root>
);
```

#### PopoverTrigger

`PopoverTrigger` is the button that triggers the `Popover`. It is also used as the anchor the `PopoverPopper` is positioned against.

`PopoverPopper` is the component that pops out in an open `Popover`.

`PopoverArrow` is the optional arrow element to render for the `Popover`. This can be used to help visually link the `PopoverTrigger` with the `PopoverPopper`. Must live inside `PopoverPopper`.
