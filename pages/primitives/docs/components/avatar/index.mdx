### Installation

Install the component from your command line.

```bash
npm install @radix-ui/react-avatar
```

Then import the component in your component file.

```jsx
// Avatar.tsx
import * as AvatarPrimitive from '@radix-ui/react-avatar';
```

### Basic example

```jsx live manual
// Avatar.tsx
// import * as AvatarPrimitive from '@radix-ui/react-avatar';

const Avatar = styled(AvatarPrimitive.Root, {
  display: 'inline-flex',
  alignItems: 'center',
  justifyContent: 'center',
  verticalAlign: 'middle',
  overflow: 'hidden',
  userSelect: 'none',
  width: 48,
  height: 48,
  borderRadius: 24,
});

const AvatarImage = styled(AvatarPrimitive.Image, {
  width: '100%',
  height: '100%',
  objectFit: 'cover',
});

const AvatarFallback = styled(AvatarPrimitive.Fallback, {
  width: '100%',
  height: '100%',
  display: 'flex',
  alignItems: 'center',
  justifyContent: 'center',
  backgroundColor: 'dodgerblue',
});

render(
  <Avatar>
    <AvatarImage src="https://picsum.photos/id/1005/400/400" />
    <AvatarFallback>UI</AvatarFallback>
  </Avatar>
);
```

#### Clickable Avatar with tooltip

```jsx live manual compact
// Avatar.tsx
// import * as AvatarPrimitive from '@radix-ui/react-avatar';
// import * as TooltipPrimitive from '@radix-ui/react-tooltip';

const Avatar = styled(AvatarPrimitive.Root, {
  display: 'inline-flex',
  alignItems: 'center',
  justifyContent: 'center',
  verticalAlign: 'middle',
  overflow: 'hidden',
  userSelect: 'none',
  width: 48,
  height: 48,
  borderRadius: 24,
  appearance: 'none',
  border: 'none',
  padding: 0,
});

const AvatarImage = styled(AvatarPrimitive.Image, {
  width: '100%',
  height: '100%',
  objectFit: 'cover',
});

const AvatarFallback = styled(AvatarPrimitive.Fallback, {
  width: '100%',
  height: '100%',
  display: 'flex',
  alignItems: 'center',
  justifyContent: 'center',
  backgroundColor: 'dodgerblue',
});

const TooltipTrigger = styled(TooltipPrimitive.Trigger, {
  backgroundColor: 'transparent',
  border: 'none',
  padding: 0,
});

const TooltipContent = styled(TooltipPrimitive.Content, {
  borderRadius: 3,
  padding: '5px 10px',
  fontSize: 14,
  backgroundColor: 'gainsboro',
  color: 'black',
});

render(
  <TooltipPrimitive.Root>
    <TooltipTrigger>
      <Avatar onClick={() => alert('Clicked!')}>
        <AvatarImage src="https://picsum.photos/id/1006/400/400" />
        <AvatarFallback>AB</AvatarFallback>
      </Avatar>
    </TooltipTrigger>

    <TooltipPrimitive.Popper side="top">
      <TooltipContent>Tooltip content</TooltipContent>
      <TooltipPrimitive.Arrow style={{ fill: 'gainsboro' }} />
    </TooltipPrimitive.Popper>
  </TooltipPrimitive.Root>
);
```

### API Reference

#### Avatar

`Avatar` is the root component.

#### AvatarImage

The image to render. By default `AvatarImage` will only render when it has actually been loaded. You can use the `onLoadingStatusChange` handler if you need finer control.

#### AvatarFallback

The fallback element. It will only render anytime `AvatarImage` hasn't loaded. This means whilst it's loading, or if there was an error. If you don't want it to flash quickly whilst it's loading, you can provide a `delayMs` prop which will delay its rendering a bit. Otherwise, if you need finer control you can use the `onLoadingStatusChange` handler on `AvatarImage`.
