### Installation

Install the component from your command line.

```bash
npm install @radix-ui/react-collapsible
```

### Anatomy

Import the components and piece the parts together.

```jsx
import {
  Collapsible,
  Button,
  Content,
} from '@radix-ui/react-collapsible';

export default () => (
  <Collapsible>
    <Button />
    <Content />
  </Collapsible>
);
```

### Basic example

Create your styled avatar component from the primitive parts.

**Note:** This example uses [Stitches](https://stitches.dev) but any styling solution is [compatible](../overview/styling).

```jsx live manual
// import { styled } from 'path-to/stitches.config';
import * as CollapsiblePrimitive from '@radix-ui/react-collapsible';

const Content = styled(CollapsiblePrimitive.CollapsibleContent, {
  padding: 10,
  marginTop: 10,
  backgroundColor: 'gainsboro',
  borderRadius: 6,
});

export default () => (
  <CollapsiblePrimitive.Root>
    <CollapsiblePrimitive.CollapsibleButton>
      Button
    </CollapsiblePrimitive.CollapsibleButton>
    <Content>Content 1</Content>
  </CollapsiblePrimitive.Root>
);
```
