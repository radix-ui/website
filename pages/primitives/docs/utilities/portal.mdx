---
title: 'Portal'
description: Useful to render a React subtree in a different part of the DOM.
features:
  - Render any React subtree outside of your App.
  - Appends to `document.body` by default but can be customised to use a different container.
---

### Installation

Install the component from your command line.

```bash
npm install @radix-ui/react-portal
```

### Anatomy

Import the components and piece the parts together.

```jsx
import { PortalPrimitive } from '@radix-ui/react-portal';

export default () => <PortalPrimitive />;
```

### Basic example

Create your portaled component from the primitive.

```jsx
export default () => (
  <>
    <Portal>Content</Portal>
  </>
);
```

### API Reference

#### Portal

`Portal` is the root component. Anything you put inside `Portal` will be rendered in a separate `<radix-portal>` element. By default, this element will be appended to `document.body` but you can choose a different container by using the `containerRef` prop.
