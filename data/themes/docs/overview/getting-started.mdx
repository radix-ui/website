# Getting started

<Description>
  Install Radix Themes and start building your project in minutes.
</Description>

## Installation

Getting up and running is quick and easy, just follow these simple steps.

### 1. Install Radix Themes

Install the package from your command line.

<Tabs defaultValue="npm" my="5">
  <TabsList>
    <TabsTrigger value="npm">npm</TabsTrigger>
    <TabsTrigger value="yarn">yarn</TabsTrigger>
    <TabsTrigger value="pnpm">pnpm</TabsTrigger>
  </TabsList>
  <TabsContent value="npm">
    <CodeBlock mt="3" language="bash" value="npm install @radix-ui/themes" />
  </TabsContent>
  <TabsContent value="yarn">
    <CodeBlock mt="3" language="bash" value="yarn add @radix-ui/themes" />
  </TabsContent>
  <TabsContent value="pnpm">
    <CodeBlock mt="3" language="bash" value="pnpm install @radix-ui/themes" />
  </TabsContent>
</Tabs>

### 2. Import the CSS file

Import the global CSS file at the root of your application.

```ts
import '@radix-ui/themes/index.css';
```

### 3. Add the Theme component

Wrap the root of your application in a `Theme` component.

```jsx line=5,7
import { Theme } from '@radix-ui/themes';

export default function () {
  return (
    <Theme>
      <MyApp />
    </Theme>
  );
}
```

### 4. Start building

You are now ready to use themed components in your app.

```jsx
import { Flex, Text, Button } from '@radix-ui/themes';

export default function MyApp() {
  return (
    <Flex direction="column" gap="2">
      <Text>Hello from Radix Themes :)</Text>
      <Button>Let's go</Button>
    </Flex>
  );
}
```

## Customizing your theme

Configuration is managed and applied via the [Theme](/themes/docs/components/theme) component.

### Basic configuration

Pass [configuration props](/themes/docs/components/theme) to `Theme` to customize the appearance.

```jsx line=1
<Theme
  __accentColor__="crimson"
  __grayColor__="sand"
  __radius__="large"
  __scaling__="95%"
>
  <MyApp />
</Theme>
```

### Using the theme panel

Previewing configuration and finding the perfect style for your application is much easier when you can visualise the changes, this is made simple by using the `ThemePanel`.

```jsx
<div>This is a live example of the theme panel.</div>
```

Simply open the panel and adjust the settings, then copy and paste the configuration.

Adding the panel to your own app is easy, import it from the package and drop it into your `Theme`:

```jsx line=7
import { Theme, ThemePanel } from '@radix-ui/themes';

export default function () {
  return (
    <Theme>
      <MyApp />
      <ThemePanel />
    </Theme>
  );
}
```
