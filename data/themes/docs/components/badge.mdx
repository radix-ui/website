## API Reference

This component is based on the `span` element and supports [common margin props](/themes/docs/theme/layout#margin-props).

<ThemesPropsTable name="badge" />

## Examples

### Size

Use the `size` prop to control the size.

```jsx live=true
<Flex gap="3" align="center">
  <Badge size="1">Badge</Badge>
  <Badge size="2">Badge</Badge>
</Flex>
```

### Variant

Use the `variant` prop to control the visual style.

```jsx live=true
<Flex gap="3">
  <Badge variant="solid">Solid</Badge>
  <Badge variant="soft">Soft</Badge>
  <Badge variant="outline">Outline</Badge>
  <Badge variant="surface">Surface</Badge>
</Flex>
```

### Color

Use the `color` prop to assign a specific color value, ignoring the global theme.

```jsx live=true
<Flex gap="3">
  <Badge color="crimson">Crimson</Badge>
  <Badge color="indigo">Indigo</Badge>
  <Badge color="grass">Grass</Badge>
  <Badge color="orange">Orange</Badge>
</Flex>
```

### High-contrast

Use the `highContrast` prop to add contrast.

```jsx live=true
<Flex direction="column" gap="3">
  <Flex gap="3">
    <Badge variant="solid">Solid</Badge>
    <Badge variant="soft">Soft</Badge>
    <Badge variant="outline">Outline</Badge>
    <Badge variant="surface">Surface</Badge>
  </Flex>
  <Flex gap="3">
    <Badge variant="solid" highContrast>
      Solid
    </Badge>
    <Badge variant="soft" highContrast>
      Soft
    </Badge>
    <Badge variant="outline" highContrast>
      Outline
    </Badge>
    <Badge variant="surface" highContrast>
      Surface
    </Badge>
  </Flex>
</Flex>
```

### Radius

Use the `radius` prop to assign a specific radius value, ignoring the global theme.

```jsx live=true
<Flex gap="3">
  <Badge radius="none">None</Badge>
  <Badge radius="small">Small</Badge>
  <Badge radius="medium">Medium</Badge>
  <Badge radius="large">Large</Badge>
  <Badge radius="full">Full</Badge>
</Flex>
```
