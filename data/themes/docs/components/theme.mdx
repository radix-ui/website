## Examples

### Basic configuration

Wrap a component tree in the `Theme` component to provide or modify configuration for all children.

```jsx live=true
<Theme __accentScale__="grass" __radius__="full">
  <Card size="2" style={{ maxWidth: 450 }}>
    <Flex direction="column" gap="4">
      <Text weight="bold">Feedback</Text>
      <TextArea placeholder="Your feedback" />
      <Button>Submit</Button>
    </Flex>
  </Card>
</Theme>
```

### Component instance overrides

Override configuration per component by passing any supported prop directly to that component.

```jsx live=true line=6
<Theme accentScale="crimson" radius="full">
  <Card size="2" style={{ maxWidth: 450 }}>
    <Flex direction="column" gap="4">
      <Text weight="bold">Feedback</Text>
      <TextArea placeholder="Your feedback" />
      <Button __color__="plum" __radius__="none">
        Submit
      </Button>
    </Flex>
  </Card>
</Theme>
```

### Nesting

Nest another theme to modify configuration for a specific subtree.

```jsx live=true line=15
<Theme accentScale="crimson" radius="full">
  <Card size="2" style={{ maxWidth: 450 }}>
    <Box>
      <Text as="div" color="gray" mb="3">
        Parent theme
      </Text>

      <Flex direction="column" gap="4">
        <Text weight="bold">Feedback</Text>
        <TextArea placeholder="Your feedback" />
        <Button>Submit</Button>
      </Flex>
    </Box>

    <Theme __accentScale__="plum" __radius__="none">
      <Card size="2" mt="6">
        <Text as="div" color="gray" mb="3">
          Child theme
        </Text>

        <Flex direction="column" gap="4">
          <Text weight="bold">Feedback</Text>
          <TextArea placeholder="Your feedback" />
          <Button>Submit</Button>
        </Flex>
      </Card>
    </Theme>
  </Card>
</Theme>
```
