---
metaTitle: Aspect Ratio
metaDescription: Displays content within a desired ratio.
---

```jsx live=true
<AspectRatio ratio={16 / 8}>
  <img
    src="https://images.unsplash.com/photo-1479030160180-b1860951d696?&auto=format&fit=crop&w=1200&q=80"
    alt="A house in a forest"
    style={{
      objectFit: 'cover',
      width: '100%',
      height: '100%',
      borderRadius: 'var(--br-2)',
    }}
  />
</AspectRatio>
```

## API Reference

This component inherits the props from the [Radix Aspect Ratio](/docs/primitives/components/aspect-ratio) primitive.
